# O1 CMS - Application Manager

## 🚀 Como Iniciar (SUPER FÁCIL!)

**Apenas execute:**
```bash
./INICIAR.sh
```

Ou dê duplo clique no arquivo `INICIAR.sh`!

O navegador vai abrir automaticamente! 🎉

---

## 📱 Acessar

- **Login**: http://localhost:8000/login.html
- **Dashboard**: http://localhost:8000/index.html
